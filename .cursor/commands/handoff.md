# /handoff — сжатие сессии

По навыку `~/.cursor/skills/handoff/SKILL.md`.

## Действия

1. Прочитать skill `handoff`.
2. Сформировать `handoffs/handoff-YYYY-MM-DD-HHmm.md`:
   - цель задачи
   - что сделано
   - что осталось
   - ключевые файлы/объекты
   - блокеры
3. Сообщить путь к handoff-файлу.
