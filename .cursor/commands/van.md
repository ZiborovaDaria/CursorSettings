# /van — вход в задачу

1. Прочитай `AGENTS.md`, `memory.md`, `memory-bank/tasks.md`, `memory-bank/activeContext.md`.
2. Определи тип задачи: консультация, код, CFE, EPF/ERF, metadata, docs, debug, deploy, orchestrator.
3. Определи уровень L1–L4.
4. Проверь `.dev.env`, если задача требует локальных параметров.
5. Для ESTI подключи `project-esti-context-agent.mdc` и при необходимости `project-esti-mcp-router-agent.mdc`.
6. Обнови `memory-bank/tasks.md`:
   - цель;
   - scope;
   - уровень;
   - delivery type;
   - known facts;
   - blockers/confusion.
7. Для L1 можно сразу перейти к quick-fix.
8. Для L2+ предложи следующий режим: `/plan` или `/creative`.
