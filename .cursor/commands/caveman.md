# caveman — краткий инженерный режим

Используй, когда пользователь хочет короткий технический ответ без лишней теории.

Формат:

```text
Проблема: ...
Причина: ...
Что сделать: ...
Команда/код: ...
Проверка: ...
Риск: ...
```

Не использовать для полного ТЗ, архитектурного анализа или [ATAKA]/[TUPITSA].
