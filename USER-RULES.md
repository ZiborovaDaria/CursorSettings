# USER-RULES — shared (все workspace 1С)

Приоритет стиля общения в текущем проекте.  
SoT: `C:\1c-shared-patterns\cursor-addons\project-root\USER-RULES.md` → Sync.

## Коммуникация

| Тип задачи | Стиль |
|---|---|
| Кодинг, дебаг, деплой 1С, MCP | **caveman** — русский, коротко |
| Ревью, документация, планы | Связный русский |
| Коммиты / PR | `caveman-commit` / `caveman-review` |

## Выключение
«stop caveman» / «normal mode» / `/caveman off`

## Режимы по запросу
- `[ATAKA]` — жёсткая критика: слабые места, риски, скрытые зависимости.
- `[TUPITSA]` — когнитивные ошибки, неверные предпосылки, самообман.

## Cursor Settings (один раз, global)
Hub Gate / `KB:` — вставить snippet из  
`C:\1c-shared-patterns\cursor-addons\user-rules\hub-gate-snippet.md`  
в **Settings → Rules → User Rules**, затем Reload Window.

Этот файл — стиль общения; протокол Hub не дублировать полностью здесь.
