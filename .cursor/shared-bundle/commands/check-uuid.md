Проверь уникальность UUID в MDO-файлах 1С по указанному пути (или текущему проекту).

Скрипт: `commands/check_uuid_duplicates.py` (рядом с этим файлом)

## Аргумент $ARGUMENTS

Пользователь может передать:
- Путь к папке или файлу для проверки
- Флаг `--fix` для автоматического исправления
- Флаг `--include-intra` для проверки внутрифайловых дубликатов
- Пустую строку — тогда проверяй текущий рабочий каталог проекта

## Выполнение

1. Определи путь для проверки:
   - Если пользователь передал путь — используй его
   - Если нет — найди папку `src/` в текущем рабочем каталоге и используй её

2. Запусти скрипт:
   ```
   py commands/check_uuid_duplicates.py <аргументы>
   ```

3. Проанализируй результат:
   - Код выхода 0 — дубликатов нет, сообщи пользователю
   - Код выхода 1 — есть дубликаты, покажи отчёт и спроси нужно ли исправить (`--fix`)

4. Если пользователь передал `--fix`:
   - Запусти с флагом `--fix`
   - Покажи что было исправлено
   - Если после исправления остались дубликаты — запусти ещё раз с `--fix` (бывает нужно 2 прохода)
