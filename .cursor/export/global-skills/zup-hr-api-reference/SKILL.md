# ЗУП 3.1 HR API reference

Полный справочник API 1С:ЗУП 3.1 для работы с кадровыми данными хранится в отдельном файле `references/zup-hr-api.md`, чтобы не раздувать контекст постоянно.

**Действие:** прочитай файл `references/zup-hr-api.md` (относительно папки скила) и используй его как справочник при решении задачи.

Файл покрывает:
- Архитектуру Сотрудник vs ФизическоеЛицо
- Иерархию модулей (КадровыйУчет, КадровыйУчетРасширенный, ЗарплатаКадры и др.)
- `КадровыеДанныеСотрудников` / `КадровыеДанныеФизическихЛиц` - основные методы и все поля
- Периоды работы, даты приема/увольнения
- Стажи (виды, расчет, представление)
- Связи Сотрудник <-> ФизическоеЛицо (основной, совместители, подработки)
- Получение списков сотрудников организации
- Временные таблицы (ВТ) для пакетных запросов
- Рабочие места, архив сотрудников
- Создание физлиц и сотрудников
- ФИО: инициалы, склонение, пол
- Текущие/плановые начисления и удержания
- Остатки отпусков
- Графики работы и производственный календарь
- Паспортные данные, штатное расписание
- Проверки кадрового учета, формирование движений
- Договоры ГПХ, совместительство
- Механизм представлений СКД (виртуальные таблицы для отчетов)
- Настройки кадрового учета, права, ответственные лица
