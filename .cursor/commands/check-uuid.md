# check-uuid — проверка UUID и metadata references

Используй после изменения metadata XML, форм, команд, ролей, подсистем, CFE metadata.

1. Найди измененные XML-файлы.
2. Проверь дубли UUID.
3. Проверь ссылки на удаленные/несуществующие объекты.
4. Проверь имена, синонимы, prefix.
5. Проверь Form.xml на битые команды/элементы.
6. При доступности — выполнить metadata validation / load check.
7. Отчет: OK/WARNING/BLOCKER.
