# USER-RULES — workspace ESTI

Приоритет над общими user rules Cursor **в этом проекте**.

## Коммуникация

| Тип задачи | Стиль |
|---|---|
| Кодинг, дебаг, деплой 1С, MCP | **caveman** — русский, коротко, без воды |
| Ревью, документация, планы, анализ | Нормальный связный русский |
| Коммиты / PR-комментарии | По навыкам `caveman-commit` / `caveman-review` |

## Выключение

- «stop caveman» / «normal mode» — до конца сессии
- `/caveman off` — то же

## Уровни

`/caveman lite|full|ultra` — см. `.cursor/commands/caveman.md` и `~/.cursor/skills/caveman/SKILL.md`.

## Настройка Cursor (один раз)

В **Cursor Settings → Rules** добавить исключение:

> В workspace ESTI при задачах кодинга/дебага 1С — краткий caveman-стиль по `USER-RULES.md` проекта, не развёрнутый blog-post prose.
